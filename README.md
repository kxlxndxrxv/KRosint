## Источники данных

### Унифицированный метод: Wayback Machine CDX API

Для обеспечения консистентности и устранения искажений данных, для всех периодов с 2000 по 2026 год используются 4 источника. Цель — собрать по 1000 никнеймов для каждой эпохи (всего 5000 ников).

Для чистоты эксперимента мы придерживаемся одной категории (IT/Технологии) на протяжении всех 25 лет.

| Сайт | Тематика | Год основания | Метод сбора |
|---|---|---|---|
| Slashdot | IT-новости | 1997 | Wayback Machine CDX API |
| SourceForge | Open-source | 1999 | Wayback Machine CDX API |
| HackerNews | IT, стартапы | 2007 | HackerNews API |
| GitHub | Разработка | 2008 | GitHub API |

### Временны́е периоды

```
2000–2004  → пик ICQ, начало Рунета, ранний веб
2005–2009  → CS 1.6, WoW, ВКонтакте запуск (2006)
2010–2014  → YouTube-культура, развитие соцсетей
2015–2019  → Twitch, Steam, e-sport бум
```

## ML-классификатор

Random Forest (200 деревьев) с 5-кратной кросс-валидацией.
Входные признаки: структурные (длина, цифры, разделители) + тематические маркеры.

Ожидаемая точность: **~30–40%** (на реальных, «грязных» данных это очень хороший показатель для ML, так как предсказать точную эпоху по одному случайному слову математически сложно).
